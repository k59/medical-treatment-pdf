# Medical Treatment PDF

该仓库保存已经打包完成的电子医疗报告静态页面，GitHub Pages 发布目录为 `docs/`。

## 开启 GitHub Pages

1. 打开仓库的 `Settings` → `Pages`。
2. 在 `Build and deployment` 中选择 `Deploy from a branch`。
3. Branch 选择 `main`，目录选择 `/docs`。
4. 点击 `Save`，等待 GitHub Pages 发布完成。
